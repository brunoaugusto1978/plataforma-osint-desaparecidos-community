# Privacidade e Ética

Este projeto lida com uma temática sensível. A versão pública existe para colaboração técnica e educacional, não para operar casos reais.

## Diretrizes

- Não publicar dados reais.
- Não publicar fotos reais.
- Não expor vítimas ou familiares.
- Não automatizar coleta sem governança.
- Não tratar dados agregados como prova individual.
- Manter revisão humana em todo fluxo de decisão.
