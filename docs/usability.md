# Usabilidade

A Community Edition simula o fluxo operacional do analista sem usar dados reais.

## Fluxo principal

1. Acessar o dashboard.
2. Consultar casos recentes.
3. Entrar no detalhe do caso.
4. Visualizar próxima ação recomendada.
5. Revisar evidências candidatas.
6. Consultar transparência por fonte em Análises OSINT.
7. Acompanhar inteligência agregada em dados fictícios.

## Princípios

- O sistema orienta o próximo passo, mas não substitui decisão humana.
- Evidências candidatas exigem revisão humana.
- Achados manuais devem ser identificados como manuais.
- Dados agregados são contexto, não prova individual.
- A versão pública não deve conter dados reais.
