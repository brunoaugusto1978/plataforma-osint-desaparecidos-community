# Plataforma OSINT para Pessoas Desaparecidas — Community Edition

Versão pública, demonstrativa e segura da Plataforma OSINT para Pessoas Desaparecidas.

Esta edição foi criada para receber colaboração em interface, usabilidade, documentação, acessibilidade, testes, componentes React e inteligência agregada com dados fictícios.

## Importante

Este repositório **não contém**:

- dados reais de pessoas desaparecidas;
- fotos reais;
- credenciais;
- Supabase real;
- migrations reais;
- Edge Functions operacionais;
- conectores reais sensíveis;
- regras completas de scoring;
- pipeline operacional privado.

## Como rodar localmente

```bash
npm install
npm run dev
```

A aplicação abre em ambiente local com dados mockados.

## Como contribuir

1. Faça um fork deste repositório.
2. Crie uma branch com sua melhoria.
3. Rode `npm run build` antes de enviar.
4. Abra um Pull Request.
5. Aguarde revisão.

## Áreas recomendadas para contribuição

- UI/UX;
- acessibilidade;
- responsividade;
- componentes React;
- documentação;
- testes;
- dados fictícios;
- visualizações de inteligência agregada;
- templates de conectores seguros.

## Regra de segurança

Nunca envie dados reais, nomes reais, imagens reais, documentos reais, URLs sensíveis ou credenciais.
