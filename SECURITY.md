# Política de Segurança

Este repositório é público e demonstrativo. Ele não deve receber informações sensíveis.

## Nunca envie

- dados reais de pessoas desaparecidas;
- fotos reais;
- documentos reais;
- nomes reais de vítimas ou familiares;
- URLs privadas;
- credenciais;
- tokens;
- `.env`;
- Supabase real;
- migrations reais;
- conectores operacionais sensíveis.

## Vulnerabilidades

Se encontrar uma falha de segurança neste repositório, abra uma issue com descrição responsável e sem expor dados sensíveis.

## Escopo

A Community Edition é focada em frontend, documentação, usabilidade e componentes demonstrativos. O core operacional privado não faz parte deste repositório.
